# Link especial de Dia dos Namorados

Abra o arquivo `index.html` no navegador.

Para personalizar pelo link, use estes parametros:

```text
index.html?nome=Maria&mensagem=Te%20amo%20muito&foto=https://exemplo.com/foto.jpg
```

- `nome`: nome que aparece na tela.
- `mensagem`: frase abaixo do nome.
- `foto`: link direto da foto.

Se quiser compartilhar pela internet, publique essa pasta em um serviço como Netlify, Vercel ou GitHub Pages.
